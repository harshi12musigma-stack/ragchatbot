                          TECHNOLOGY STACK SUMMARY

  Layer              Technology                  Justification
  ─────────────────  ──────────────────────────  ────────────────────────────
  Frontend           React 18 + Vite             Modern, fast, large ecosystem
  Styling            TailwindCSS                 Utility-first, rapid proto
  Backend            FastAPI (Python 3.11+)      Async, LLM-friendly
  Vector DB          ChromaDB                    Lightweight, Python-native
  Embeddings         text-embedding-3-small      Cost-effective, 1536-dim
  LLM                gpt-4o-mini                 Fast, accurate, affordable
  Orchestration      LangChain                   RAG patterns, templates
  File Parsing       PyPDF2, python-docx         Standard, reliable
